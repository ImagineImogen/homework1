## Python Web development course homework

###### *This script lists most frequently used verbs in the function names and their occurrence in the python files in the given python projects directories*

**Installing:**

Make a new virtualenv and activate it

```$ virtualenv -p python3 homework1```

```$source /bin/activate```

Install requirements

```$ pip install -r requirements.txt```


**How to use:**

*place the script in the same directories as your python projets and run it:*

```$ cd mostly_used_verbs.py```

```$ python mostly_used_verbs.py```
